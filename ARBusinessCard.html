Business card application
  • Internal project for internship company
  • Uses Vuforia Implementation for image tracking
  • I worked on:
    • The basic AR setup
    • The animation controller
    • The character's head and body movements
    • The text bubble
This application was a Project made internally for Movares, which is the company where I did my first ever internship. The application was initially made to be a Modifiable AR presentation for a person internally at Movares. We later on updated some of the less flexible systems to create an easy to use template for potential future cases.
For this project I set up the project and image target tech together with the other developer, I made a script that handled the flow and order of the animations that played and associated text, I made sure the character always stayed rotated towards the user with both body and head, and I set up the text bubble to display the intended text at a variable speed.
